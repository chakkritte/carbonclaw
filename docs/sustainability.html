Carbon Tracking

Sustainability First

CarbonClaw is the first AI agent runtime built with environmental impact as a core metric. We use the codecarbon library to monitor and report the CO2 emissions of every task.

Real-Time Monitoring

While you chat, CarbonClaw tracks the power consumption of your local hardware and estimates the emissions from cloud network calls. Each response turn ends with a detailed metric:

[Carbon Emissions: 0.000015 kg CO2]

Green Recommendations

When the Smart Router detects a simple task, it will proactively suggest using a local model to reduce your footprint:

🌱 "This task seems simple. Consider using a local LLM (e.g., Llama 3.2 via Ollama) to reduce carbon footprint from network and cloud compute."

Sustainability Reports

You can view your project's total environmental impact at any time using the /carbon command:

/carbon

This command provides a comprehensive report, including your total emissions and an equivalent "driving distance" comparison to help put the data in context.

Offline Mode

For air-gapped or low-connectivity environments, CarbonClaw can run in Offline Mode, using regional emission averages instead of live cloud data.
