Documentation Overview

Welcome to the CarbonClaw documentation. CarbonClaw is an AI-native runtime designed for professional, sustainable, and private autonomous engineering.

Local-First Philosophy

CarbonClaw prioritizes local LLMs (via Ollama) to ensure data privacy and reduce the environmental impact of cloud computation.

Project Principles

  • Privacy-First: Sensitive code never leaves your machine unless you explicitly use a cloud provider.
  • Sustainability: Real-time CO2 monitoring helps you understand the impact of your AI usage.
  • Autonomous: Go beyond simple chat. Let specialized agents coordinate to solve complex engineering cycles.
