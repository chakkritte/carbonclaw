Slide Generation

CarbonClaw can autonomously design and generate professional PowerPoint presentations by leveraging the PptxGenJS Node.js library.

How it Works

The slide generation workflow uses a hybrid Python-Node.js execution model:

  1. Instruction: You ask the agent to create a deck (e.g., "Create a 5-slide presentation on quantum computing").
  2. Code Generation: The agent writes Node.js code that uses the PptxGenJS API to build slides, text, and charts.
  3. Execution: CarbonClaw uses the run_nodejs tool to install dependencies (if needed) and execute the script in a secure temporary directory.
  4. Delivery: The resulting .pptx file is saved to your local directory.

Usage Example

> Create a title slide and one content slide about CarbonClaw sustainability.

The agent will automatically route this to the SLIDES task type and utilize the specialized Node.js tool.

Security & Approval

Node.js execution requires your approval by default. You can see the code the agent intends to run before it executes.

Specialized API Context

CarbonClaw agents are pre-loaded with a concise PptxGenJS API reference, including master slides, background styling, and layout coordinates, ensuring high-quality output every time.
